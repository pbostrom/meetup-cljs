# meetup-cljs

Austin Clojure Meetup Clojurescript demo

## Prerequisites

You will need [Leiningen][1] 2 installed.

[1]: https://github.com/technomancy/leiningen

## Running
Build the Clojurescript:

    lein cljsbuild once

To start a web server for the application, run:

    lein ring server
